# Regionales Mia Jazmin

Tienda online de productos regionales artesanales de La Rioja, Argentina.

## ✨ Características
- 📱 Diseño Responsive (Adaptable a móviles y tablets).
- 🧀 Catálogo de productos organizado por categorías.
- 🛒 Integración directa con WhatsApp para pedidos.
- 🚀 Optimizado para SEO y performance.

## 🛠️ Tecnologías
- HTML5 semántico.
- CSS3 moderno (Flexbox, Grid, Variables).
- JavaScript Vanilla (Lazy loading, UX).
